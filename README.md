# Factory OS Agent

An open-source Manufacturing ERP / Factory OS agent for streamlining production workflows. This project explores modernizing factory operations by providing a modular, API-first system to manage manufacturing processes, inventory, and quality control.

## Key Features
- Real-time production tracking and job scheduling
- Inventory management with low-stock alerts
- Quality control checkpoints and defect logging
- Basic reporting dashboard for operational metrics

## Tech Stack
- Node.js / Express.js backend
- React frontend
- PostgreSQL database
- Docker for containerization

## Getting Started
```bash
git clone https://github.com/zoreanuj/factory-os-agent.git
cd factory-os-agent
npm install
npm run dev
```
Configure your `.env` file with database credentials and visit `http://localhost:3000`.
